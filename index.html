Instructions on how to use winget

Syntax:

winget install ""

winget search ""

winget uninstall ""

winget source ""

winget upgrade ""

winget list ""

winget hash ""

winget export ""

winget validate ""

winget settings ""

winget import ""

winget show ""

winget pin ""

winget configure ""

winget download ""

winget repair ""

|

What they do:

winget install: Installs the named package

winget search: searches for packages with the given name as either a close name or tag

winget uninstall: Uninstalls named package

winget source: Manages sources of packages

winget upgrade: Updates named package

winget list: Shows installed packages

winget hash: Helper to hash installer files

winget export: Exports a list of the installed packages

winget validate: Validates a manifest file

winget settings: Open settings or set administrator settings

winget import: Installs all the packages in a file

winget show: Show information about named package

winget pin: Manage package pins

winget configure: Configures the system into a desired state

winget download: Downloads the installer from a named package

winget repair: Repairs the selected packages
